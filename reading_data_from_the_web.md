## SW data

``` r
url = "https://www.imdb.com/list/ls070150896/"
swm_html = read_html(url)
```

How do I get the stuff that I want?

``` r
sw_titles = 
  swm_html %>%
  html_elements(".lister-item-header a") %>%
  html_text()

sw_runtime = 
  swm_html %>%
  html_elements(".runtime") %>%
  html_text()

sw_money = 
  swm_html %>%
  html_elements(".text-small:nth-child(7) span:nth-child(5)") %>%
  html_text()

sw_df = 
  tibble(
    title = sw_titles,
    runtime = sw_runtime,
    money = sw_money
  )
```
